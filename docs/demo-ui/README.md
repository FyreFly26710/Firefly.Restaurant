# Demo UI Reference

This folder contains prototype UI references for the Firefly Restaurant project.

Use these files for:

- visual direction,
- interaction ideas,
- layout and content-management inspiration,
- initial UX discussion.

Do not use these files as:

- production application code,
- canonical product copy,
- canonical restaurant identity,
- API contract definitions,
- database seed data.

The sample restaurant identity in this folder is reference-only and should not be treated as Firefly Restaurant product content.
